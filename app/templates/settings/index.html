Gmail / Google Workspace — Google no longer accepts account passwords for SMTP. You need an App Password, which requires 2-Step Verification turned on first. If the App passwords option is missing from your Google account, 2-Step Verification is not enabled — enable it, then revisit the App Passwords page.

Outlook / Office 365 / Hotmail — Same restriction. Enable 2-Step Verification and create an App Password at account.microsoft.com → Security → Advanced options → App passwords.

Yahoo — Account Key / 2-Step Verification required, then generate an app password.

Easiest alternative — Use a transactional email provider (Mailgun, SendGrid, Brevo, Postmark, Amazon SES, etc.) or your ISP's SMTP relay. These accept ordinary username/password credentials and typically use port 587 with TLS.
